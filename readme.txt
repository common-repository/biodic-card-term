== Description ==

BioScripts - Centro de Investigación y Desarrollo de Recursos Científicos te trae una forma fácil de mostrar una definición de una palabra definida en el proyecto BioDic, un diccionario de términos científicos sencillo.


== Installation ==
1. Sube el archivo a wp-content/plugin
2. Activar plugin
3. En tus entradas usa el widget [biodic name="hoja"]
5. &iexcl;Comprueba que se ve correctamente!
6. &iexcl;Dale un poco de diseño!
